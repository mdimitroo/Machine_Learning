# Images Directory

Place images related to the bias-variance tradeoff concept here.

## File Naming Convention

Use descriptive names with underscores:
- `bias_variance_tradeoff.png`
- `bias_variance_visualization.png`
- `error_components_diagram.png`

## Supported Formats

- PNG (recommended for diagrams)
- JPG/JPEG (for photos)
- SVG (for scalable vector graphics)

## Usage in README

Reference images using relative paths:
```markdown
![Alt text](images/filename.png)
```

## Citation

Always cite the source of images in the main README's References section.
